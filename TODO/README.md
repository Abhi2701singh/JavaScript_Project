# 📝 TODO App  

A simple and interactive **TODO App** built with JavaScript, HTML, and CSS. Easily add, manage, and remove tasks to stay organized.  

---

## 🚀 Features  
✅ Add new tasks  
✅ Mark tasks as completed  
✅ Delete tasks  
✅ Persistent storage with localStorage
